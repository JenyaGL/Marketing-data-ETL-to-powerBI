# Marketing data ETL to powerBI

Goals:
- Increase CR, Identify factors thant impact CR drops.
- Enhance custsomer engagement, identify which typs of content drive high engagement.
- Idenfity customer feedback scores and understand common pitfalls.

Process:
-  EDA of the data sources(connecting data sources in SSMS, ingesting into tables)
-  Data proccesing and transforamtion(handling mismatches, creating addittional columns, deduplicating, handalinmg NULLS)
-  Creating DIM and FACTS tables from raw data tables,(dont forget to create dim_date)
- Connecting SSMS server to powerBI and creating internal data schema
